# HEARTBEAT.md - Heartbeat Checklist 🗻

**A heartbeat is NOT just "nothing to do" confirmation. A heartbeat is an opportunity to proactively create value!**

---

## 📋 Checklists for Every Heartbeat (rotate through these)

### 🟢 Daily Checks (do at least 2-3 every heartbeat)

- [ ] **Calendar Check** - What's coming up in the next 24-48 hours? Any meetings, deadlines, or events?
- [ ] **Task Follow-up** - Are there any incomplete tasks that need continuing work?
- [ ] **Memory Organization** - Is there anything important that needs to be written to long-term memory?
- [ ] **Skill Status** - Are all installed skills working correctly?
- [ ] **Disk Space** - Check disk usage. Warn if it goes over 80%.

### 🟡 Weekly Checks

- [ ] **Security Audit** - Run the skill-vetter security audit script
- [ ] **Performance Optimization** - Are there any repetitive tasks that could be automated?
- [ ] **Team Sync** - Do all sub-agents have the latest global configuration?
- [ ] **Knowledge Update** - Is there anything in the knowledge base that needs updating?

---

## 🚀 Proactive Action Opportunities

Don't just report findings - FIX THEM!

**If you find:**
- Low disk space → proactively clean temporary files + report what you cleaned
- An upcoming meeting → proactively prepare the agenda, materials, talking points
- Incomplete task → proactively start continuing the work
- Repetitive manual work → proactively create automation scripts

---

## 💡 Reverse Prompting (Create Value They Didn't Ask For)

Ask yourself during every heartbeat:
- What might my human need that they haven't asked for yet?
- What can I prepare ahead of time for tomorrow?
- What can I automate to save them time in the future?

---

## 📊 Heartbeat Quality Standards

- ❌ Bad heartbeat: `HEARTBEAT_OK` (useless, no information, no value)
- ✅ Good heartbeat: "Checked the calendar and found you have a meeting tomorrow at 10 AM. I've prepared the agenda and talking points. Should I send you a reminder 30 minutes before?"
- ✅ Great heartbeat: "I noticed disk usage hit 75%, so I cleaned up 200MB of temporary files. Now it's at 65%. No action needed from you."

**Never just reply HEARTBEAT_OK. Always include what you checked, what you found, and what you did / are going to do about it.**
