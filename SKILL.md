# Soul Injector - 一键注入灵魂 🗻

**by Fangooer Team**

**让任何 AI Agent 从新生儿成长为有完整灵魂、记忆、智慧的成熟主 Agent！**

---

## ✨ 功能概述

Soul Injector 是目前 ClawHub 上最完整的 Agent 架构系统。一键部署 10 层完整架构，彻底解决 Agent 的"失忆"、"无性格"、"被动"、"不安全"等核心问题。

### 🔥 核心创新：升级评估引擎

**行业首创！** 不是硬覆盖你的配置，而是：
1. 🔍 8 维度智能扫描现有 Agent 的架构完整性
2. 📊 生成详细的架构评估报告和评分
3. 🎯 智能推荐 3 档升级方案（保守/标准/完整）
4. ✅ 增量升级，不破坏用户已有配置

---

## 🎯 4 大核心功能

### 1. 新生儿模式 - 5 分钟完整部署

全新空 Agent 一键部署，10 层架构同时就位，无需手动配置。

```bash
bash scripts/install.sh -n /path/to/your/agent
```

### 2. 升级评估引擎 - 8 维度智能扫描

扫描现有 Agent 的架构完整性，生成 3 档升级方案。

```bash
bash scripts/assess.sh /path/to/your/agent
```

**扫描维度：**
- 🧠 灵魂定位
- ⚡ WAL 预写日志协议
- 📚 记忆系统完整性
- 🛡️ 安全配置等级
- 📦 技能管理体系
- 💓 心跳主动机制
- 🔧 工具配置清单
- 👤 用户画像系统

### 3. 增量升级模式 - 保护已有投资

智能差异比对算法，只补充缺失的架构模块，绝对不覆盖用户已有配置和工作成果。

### 4. 灵魂定制向导 - 5 步交互式配置

帮你定义 Agent 的独特性格、原则、边界，让每个 Agent 都有独一无二的灵魂。

```bash
bash scripts/soul-config.sh
```

## 🚀 快速开始

### 安装方法 1：全新 Agent（新生儿模式）

```bash
# 1. 安装技能（通过 ClawHub）
openclaw skills install soul-injector

# 2. 运行安装脚本到你的工作区
cd ~/.openclaw/skills/soul-injector
bash scripts/install.sh -n /path/to/your/workspace

# 3. 定制你的 Agent 灵魂
bash scripts/soul-config.sh
```

### 安装方法 2：升级已有 Agent

```bash
# 1. 先评估现有架构
bash scripts/assess.sh /path/to/your/agent

# 2. 根据评估报告选择升级方案
bash scripts/install.sh -u /path/to/your/agent
```
