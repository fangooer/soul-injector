## ✨ 什么是 Soul Injector？

Soul Injector 是一个完整的 Agent 架构系统，一键部署 10 层完整架构，让你的 Agent 立刻拥有：

- 🧠 **完整灵魂** - 性格、原则、边界
- ⚡ **预写日志系统** - WAL 协议，永不遗忘关键信息
- 📦 **上下文保护** - 工作缓冲区，对抗压缩失忆
- 🛡️ **三层安全防御** - 安全审查、注入防护、泄露防护
- 👥 **子 Agent 管理** - 完整的团队协作能力
- 💓 **心跳主动机制** - 主动发现和解决问题
- 📚 **长期记忆系统** - 永久记忆 + 向量库检索
- 🚀 **10 层完整架构** - 从灵魂到操作系统的完整体系

## 🎯 核心功能

### 1. 新生儿模式 - 一键部署

```bash
# 5 分钟完成完整架构部署
bash scripts/install.sh -n /path/to/your/agent
```

### 2. 升级评估引擎 - 核心创新

**8 维度智能扫描，3 档升级方案供选择：**

```bash
# 扫描已有 Agent 的架构完整性
bash scripts/assess.sh /path/to/your/agent
```

**扫描维度：**
- 灵魂定位
- WAL 预写日志
- 记忆系统
- 安全配置
- 技能管理
- 心跳机制
- 工具配置
- 用户画像

### 3. 增量升级 - 不破坏已有配置

智能差异比对，只补充缺失的模块，保护你已有的工作成果。

### 4. 灵魂定制向导 - 交互式配置

```bash
# 5 步交互式定义你的 Agent 的独特灵魂
bash scripts/soul-config.sh
```

## 🚀 快速开始

### 方法 1：全新 Agent 部署（新生儿模式）

```bash
# 1. 下载或克隆本项目
# 2. 运行安装脚本
bash scripts/install.sh -n /path/to/your/workspace

# 3. 定制你的 Agent 灵魂
bash scripts/soul-config.sh
```

### 方法 2：升级已有 Agent

```bash
# 1. 先评估现有架构
bash scripts/assess.sh /path/to/your/agent

# 2. 根据评估结果选择升级方案
bash scripts/install.sh -u /path/to/your/agent
```
