## 📦 Installation / 安装方式

### Option 1: GitHub Release (Recommended) / GitHub 发布版（推荐）
```bash
curl -L https://github.com/fangooer/soul-injector/releases/download/v1.0.0/soul-injector-v1.0.0.tar.gz -o soul-injector.tar.gz
tar -xzvf soul-injector.tar.gz
cd soul-injector
bash scripts/install.sh -n /path/to/your/agent
```

### Option 2: Git Clone / Git 克隆
```bash
cd ~/.agents/skills/
git clone https://github.com/fangooer/soul-injector.git
```

### Option 3: Local Install / 本地安装
Just copy the folder to your OpenClaw skills directory:
```bash
cp -r soul-injector ~/.agents/skills/
```
