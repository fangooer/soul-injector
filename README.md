## 🚀 Quick Start / 快速开始

### Prerequisites / 前置要求
- Bash shell (Linux/macOS)
- Your Agent workspace directory

### Installation / 安装

```bash
# 1. Download the latest release / 下载最新版本
curl -L https://github.com/fangooer/soul-injector/releases/download/v1.0.0/soul-injector-v1.0.0.tar.gz -o soul-injector.tar.gz

# 2. Extract / 解压
tar -xzvf soul-injector.tar.gz
cd soul-injector

# 3. Newborn Mode - Deploy to a new Agent / 新生儿模式 - 部署到新 Agent
bash scripts/install.sh -n /path/to/your/agent

# 4. OR Assessment Mode - Scan existing Agent / 或评估模式 - 扫描现有 Agent
bash scripts/assess.sh /path/to/your/agent

# 5. Run tests / 运行测试
bash run_tests.sh
```
